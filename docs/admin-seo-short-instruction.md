# SEO Admin: короткая инструкция

Адрес:

```text
https://brkovic.ltd/admin-seo.html
```

Вход общий с остальной админкой. Если уже вошли в Post Admin или Comment Admin, SEO Admin откроется без повторного пароля.

## Основной порядок

1. Откройте `Admin panel`.
2. Выберите `SEO Admin`.
3. Нажмите `Серверный аудит`.
4. Посмотрите карточки `Готово`, `Нужно проверить`, `Ошибки`.
5. В списке страниц откройте страницу с предупреждением.
6. В блоке `Детали` смотрите конкретную причину.

## Кнопки

`Серверный аудит`  
Полный backend-аудит: sitemap, robots, title, description, canonical, hreflang, OG/Twitter, JSON-LD.

`Sitemap`  
Проверяет карту сайта и наличие языковых ссылок.

`Robots`  
Проверяет, что сайт открыт для поисковиков, а админки закрыты.

`Карта URL`  
Показывает список публичных страниц для контроля `sitemap.xml`.

`IndexNow key`  
Создает или проверяет key-файл в корне сайта. Это нужно для быстрой отправки URL в Bing/Yandex.

`Отправить IndexNow`  
Отправляет публичные URL через IndexNow. Это не Google.

`AI черновик`  
Создает SEO-черновик через серверный OpenAI-ключ. AI ничего сам не публикует.

`Копировать ТЗ`  
Копирует подготовленное AI-задание.

`Сохранить карту`  
Сохраняет список полученных идентификаторов на сервере как рабочую карту. OpenAI-ключ здесь не хранится.

## Режимы экрана

Desktop: три рабочие зоны — страницы, консоль, детали.  
Tablet: блоки складываются в одну колонку, список страниц идет сеткой.  
Mobile: все элементы идут одной колонкой, кнопки растягиваются на ширину экрана.

## Дисциплина

SEO Console не является произвольным серверным терминалом. Это рабочая машина разрешенных сценариев: аудит, настройки, IndexNow и AI-черновик. Она не выполняет произвольные shell-команды и не публикует AI-текст без ручного принятия.
