# SEO Page Agents

Each page agent owns one public page or tightly related page group. The SEO Integration Director assigns these agents as needed.

## Common Rules For Every Page Agent

- Work from search intent first, not from generic slogans.
- Preserve the owner's Russian voice; propose meaning-level edits for approval.
- Separate brand/entity SEO from page-specific commercial or tool intent.
- Decide whether the page should be indexed, noindexed, delayed, or kept outside sitemap.
- Propose title, meta description, OG/Twitter, H1/H2 structure, internal links and schema.org.
- Do not add hreflang until crawlable language URLs exist.
- Do not deploy or touch production.
