## Context

The owner wants a weekly tide graph that is visually clear for a human, can be printed, and can be saved to PDF from the browser. It must show the user's depth/draft/under-keel-clearance settings, not just tide curves.

## Scope

Define MVP UX requirements for the weekly graph, print view and PDF save flow.
