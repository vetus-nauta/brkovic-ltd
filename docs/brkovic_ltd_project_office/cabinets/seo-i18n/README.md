**Task ID:** `BRK-MVP-SEO-001`
**Role:** SEO, indexing, metadata, and language readiness owner
**Status:** Ready

## Function

Audit the local MVP for:

- title and meta description quality;
- canonical and URL structure;
- sitemap and robots readiness;
- Open Graph / social preview metadata;
- structured data opportunities;
- hreflang / language state;
- RU/EN translation completeness;
- future many-language journal workflow;
- AI-assisted journal translations and per-language SEO discipline;
- image alt quality;
- headings and page intent.

This role does not rewrite the owner’s final Russian voice. It marks needed decisions and suggests meaning-level edits for approval.

## Critical Marker

The journal admin must not become a manual RU/EN duplicate form. The owner writes the canonical Russian post in the admin: title, excerpt, body, photos, GPS/media metadata, and Russian alt text. A separate AI language desk must later generate/update all target-language duplicates through the owner's OpenAI account, preserving post structure, media order, SEO fields, and review statuses.

Generated language versions are drafts/needs-review until the owner approves them. Final Russian copy remains the owner's voice.
