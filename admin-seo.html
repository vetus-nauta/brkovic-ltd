Короткая инструкция

Сначала нажмите Серверный аудит. Если есть красные пункты, откройте страницу в списке и смотрите подсказку. Идентификаторы сохраняются на сервере, IndexNow умеет создать key-файл, AI возвращает только черновик без автопубликации.

  • Серверный аудит — общий аудит страниц, sitemap и robots на backend.
  • Карта URL — показывает публичные URL для контроля карты сайта.
  • IndexNow key — создает key-файл в корне сайта.
  • Отправить IndexNow — отправляет URL в IndexNow для Bing/Yandex.
  • AI черновик — готовит SEO-рекомендации через OpenAI без автопубликации.
